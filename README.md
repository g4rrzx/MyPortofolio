A modern, responsive, and high-performance personal portfolio website built for a Web3 Developer and Researcher. The design features a minimalist yet futuristic aesthetic with dark mode by default, glassmorphism effects, and smooth animations using Framer Motion.

## 🌟 Key Features

- **Hero Section**: An engaging entry point with interactive 3D-like orbit rings and dynamic text animations highlighting a focus on the decentralized future.
- **About Me**: A section detailing my background in Computer Science and passion for decentralized systems.
- **Tech Stack & Skills**: A visually appealing grid showcasing expertise in Solidity, React, Python, Smart Contracts, Rust, Node.js, Web3.js, IPFS, and more.
- **Projects Showcase**: A dedicated area for both general web development projects and specialized Web3 Research/dApps.
- **Contact Integration**: Easily accessible contact links for networking and collaboration.

## 🛠️ Built With

- **[React 18](https://reactjs.org/)**: Core UI Library
- **[Vite](https://vitejs.dev/)**: Next Generation Frontend Tooling (Fast build & HMR)
- **[Tailwind CSS v4](https://tailwindcss.com/)**: Utility-first CSS framework for rapid styling
- **[Framer Motion](https://www.framer.com/motion/)**: For smooth, declarative animations perfectly suited to the futuristic Web3 theme
- **[Lucide React](https://lucide.dev/)**: Clean, consistent icon set

## 🚀 Getting Started Locally

To get a local copy up and running, follow these simple steps.

### Prerequisites

Ensure you have Node.js and npm installed on your local machine.

### Installation

1. **Clone the repo**
   ```bash
   git clone https://github.com/g4rrzx/MyPortofolio.git
   ```
2. **Navigate into the directory**
   ```bash
   cd MyPortofolio
   ```
3. **Install NPM packages**
   ```bash
   npm install
   ```
4. **Run the development server**
   ```bash
   npm run dev
   ```
5. Open your browser and visit `http://localhost:5173` to view the local site.
